# benfordslaw.cr

A Crystal shard which provides a check if a number array confirms to the Benford's Law

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     benfordslaw:
       github: cmiksche/benfordslaw.cr
   ```

2. Run `shards install`
